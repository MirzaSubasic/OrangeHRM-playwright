## Project description 

This is learning project aimed to practice playwright on a demo page. Code that was writen in this repository used best practices like POM (page object model) and fixtures for global set up and tear down.

## Running the tests

After pulling project run following commands on terminal

To install playwright -  **npm init playwright@latest**

To run tests -  **npx playwright test**

Aditional CLI commands can be found on https://playwright.dev/docs/test-cli
